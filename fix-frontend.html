Временное исправление фронтенда

Проблема: фронтенд подключается к localhost:4000 вместо сервера

Решение через веб-консоль провайдера:

  1. Зайдите в веб-консоль вашего VPS провайдера
  2. Выполните команды:
cd /root/Krot_Party
git pull
docker compose up -d --build frontend

Или вручную обновите docker-compose.yml:

Замените строку:

- VITE_API_BASE=http://backend:4000

На:

- VITE_API_BASE=http://93.115.203.113:4000

Затем выполните:

docker compose up -d --build frontend

Проверка:

После обновления откройте: http://93.115.203.113:5173
